# Multivariate Probability

This project covers multivariate probability concepts including mean, covariance, correlation, and the multivariate normal distribution.

## Files

- `0-mean_cov.py` - Calculate the mean and covariance of a data set
- `1-correlation.py` - Calculate a correlation matrix from a covariance matrix
- `multinormal.py` - Multivariate Normal distribution class with PDF calculation

## Requirements

- Python 3.5
- NumPy 1.15
- pycodestyle 2.5
