# Bayesian Probability

This project covers Bayesian probability concepts including likelihood, intersection, marginal probability, and posterior probability.

## Files

- `0-likelihood.py` - Calculate likelihood of obtaining data given hypothetical probabilities
- `1-intersection.py` - Calculate intersection of obtaining data with hypothetical probabilities
- `2-marginal.py` - Calculate marginal probability of obtaining data
- `3-posterior.py` - Calculate posterior probability given data

## Requirements

- Python 3.5
- NumPy 1.15
- pycodestyle 2.5
