# Linear Algebra Scripts

This folder contains Python scripts and modules for basic linear algebra operations, including:

- Array and matrix slicing
- Matrix shape calculation
- Matrix transposition
- Element-wise and matrix operations (addition, subtraction, multiplication, division)
- Concatenation of arrays and matrices
- Matrix multiplication (manual and with NumPy)
- NumPy array manipulations

## File List

- `0-slice_me_up.py`: Array slicing examples
- `1-trim_me_down.py`: Extracting columns from a matrix
- `2-size_me_please.py`: Calculate the shape of a matrix
- `3-flip_me_over.py`: Transpose a 2D matrix
- `4-line_up.py`: Element-wise addition of arrays
- `5-across_the_planes.py`: Element-wise addition of 2D matrices
- `6-howdy_partner.py`: Concatenate two arrays
- `7-gettin_cozy.py`: Concatenate two 2D matrices along a specific axis
- `8-ridin_bareback.py`: Matrix multiplication (manual)
- `9-let_the_butcher_slice_it.py`: Advanced slicing with NumPy
- `10-ill_use_my_scale.py`: Get the shape of a NumPy array
- `11-the_western_exchange.py`: Transpose a NumPy array
- `12-bracin_the_elements.py`: Element-wise operations with NumPy
- `13-cats_got_your_tongue.py`: Concatenate NumPy arrays
- `14-saddle_up.py`: Matrix multiplication with NumPy

## Usage

Run any script using Python 3:

```sh
python3 0-slice_me_up.py
```

Some scripts require NumPy. Install it with:

```sh
pip install numpy
```
